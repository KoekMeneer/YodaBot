Yoda discord bot

A Yoda themed Discord bot with a number of fun commands and the ability to respond to user chat messages.

Invite Vote

Features

A number of fun commands

The Yoda bot has a number of fun / meme commands that can be used.

You can find all these commands on our webiste or by using the &help command in your server.

Responds to messages

As you talk in your server the Yoda bot may respond to certain messages.

How it works is that the bot will search for certain keywords in the user messages and if it found for example `hmmm` it will create a response (in this case `hmmmm`) and send this in the channel where the message was found.
